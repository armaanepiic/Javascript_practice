# JavaScript Roadmap

An interview-focused learning path for frontend and full-stack JavaScript roles.
Learn the topics in order, because each phase builds on the previous one and
matches how interviewers usually evaluate candidates.

These are mandatory beginner topics. You should be able to answer basic
interview questions from this section confidently.

1. Variables (`var`, `let`, `const`)
2. Data types
   - String
   - Number

   - Null
   - Undefined
   - Symbol
   - BigInt
3. Operators
4. Type conversion
5. Truthy and falsy values
6. Conditional statements
   - `if` / `else`
   - `switch`
7. Loops
   - `for`
   - `while`
   - `do...while`
8. Functions
   - Function declaration
   - Function expression
   - Arrow function
9. Scope
   - Global scope
   - Function scope
   - Block scope
10. Arrays
11. Objects
12. Template literals

## Best Complete Learning Path

For maximum interview ROI:

1. Complete JavaScript.info
2. Watch Namaste JavaScript
3. Read MDN documentation
4. Study Frontend Interview Handbook
5. Build 30 JavaScript projects
6. Practice polyfill implementations
7. Do mock interviews

To be strongly interview-ready, complete topics 1-130. For most junior and
mid-level jobs, mastering topics 1-95 plus building 5-10 projects is usually
enough to outperform most candidates.
